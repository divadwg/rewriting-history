## UI Override
- Design system: See `/design.md` for the full "Scholarly Editorial" design system
- Primary: `#a23f00` (terracotta, not orange)
- Background/surface: `#f9f9f9` (not white)
- Fonts: Newsreader (serif) for headings, Work Sans for body, DM Mono for data
- NO 1px solid borders — use background color shifts or ghost borders at 15% opacity
- NO heavy shadows — use tonal layering instead
- Glassmorphism for nav: `background: rgba(249,249,249,0.8); backdrop-filter: blur(12px)`
